###### StackUs

### StackUs 란? 
---
+ 온라인 상에서 교사와 학생들 간에 소통과 학습을 동시에 할 수 있는 플랫폼입니다.

### 사용자
---
+ 학교에서 직접 학생들을 가르치는 교사부터 과외를 통해 몇명의 학생을 가르치는 소규모 교사들도 쉽게 문제를
 내고, 학생관리와 피드백을 할 수 있는 플랫폼 입니다.
